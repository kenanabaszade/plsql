PL/SQL

PL/SQL (Procedural Language/Structured Query Language) Oracle verilənlər bazası ilə işləmək üçün istifadə olunan prosedur proqramlaşdırma dilidir. Bu dil SQL sorğularını prosedur xüsusiyyətlərlə birləşdirərək daha funksional və effektiv proqramlar yazmağa imkan verir.

Bu kitab Oracle mühitində prosedur dilinin (PL/SQL) əsaslarına dərindən yiyələnmək istəyənlər üçün nəzərdə tutulmuşdur. PL/SQL dilində funksiyalar, prosedurlar, triggerlər, paketlər və daha çoxunu effektiv şəkildə öyrənmək üçün nəzəri və praktik misallar təqdim olunur.
Həm yeni başlayanlar, həm də təcrübəli proqramçılar üçün hazırlanmış bu resurs sayəsində siz məlumat bazası əməliyyatlarını daha təhlükəsiz və sürətli şəkildə həyata keçirə biləcəksiniz. Oracle 19C mühitinə uyğunlaşdırılmış kitab, PL/SQL ilə bağlı aktual yenilikləri və real-case nümunələri özündə cəmləşdirir.
